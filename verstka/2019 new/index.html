Каруселька навыков

Для ускорения загрузки ресурсов применены
  1. Предзагрузка стилей, для браузеров с поддержкой.
  2. Асинхронная загрузка скриптов, в порядке очереди.
  3. Ленивая загрузка для картинок.
  4. Правильные глифы нужных шрифтов (woff2).
Используемые вещи
  • Bootstrap - для быстрой верстки.
  • SASS предпроцессор - для настройки сетки, отступов, цветов и удаления лишнего функционала.
  • Jquery - для манипуляций с DOM и как основа для плагинов.
  • Snap.svg - для оживления svg.
  • Owl carousel - неплохая UI каруселька, с тачем и подстройкой кал-ва слайдов под разрешение.
  • А так же, фиксированное меню с анимацией, через клонирование основного и анимированная кнопка вверх.
